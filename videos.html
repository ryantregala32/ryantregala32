Instructional Demonstration

This video demonstrates a step-by-step sequence for creating a triangle using Snap!, a block-based programming language. The demonstration emphasizes sequencing, repetition, and computational thinking fundamentals.

Skills Demonstrated

  • Instructional design and sequencing
  • Beginner-focused technical communication
  • Computational thinking
  • Explaining programming and problem-solving visually

Demonstration titled “Show What You Know Assignment”, an exercise designed to help learners reflect on and apply their knowledge.

Demonstration titled “Teaching Programming Portfolio Lesson Plan”, a lesson plan designed to help people on scratch using the animal parade.

Demonstration titled “Nested Repeat on Snap!”, a lesson plan designed to make a mouse create a octagon using nested loops on Snap.

Demonstration titled “Monkey Banana game”, a lesson plan designed using snap to use different parts such as broadcast and when I receive.

Demonstration titled “EDS 124BR Week 4 Show What You Know”, a lesson plan designed using snap to use different parts such as the repeats and variables to make shapes.

Demonstration titled “Conditional Loop EDS 124BR”, a lesson plan designed using snap to use conditional loops.

Demonstration titled “EDS 124BR If-Else Program”, a lesson plan designed using snap to use an if-else statement.

Demonstration titled “Nested If Else Statement for EDS 124BR”, a lesson plan designed using snap to use an nested if-else statement.

Demonstration titled “Compound Conditionals”, a lesson plan designed using snap to compare and contrast conditions using and, or, and if and else.

Demonstration titled “EDS 124BR Abstraction”, a lesson plan designed using snap to make a picture using abstraction.
